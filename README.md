# crime_resolution_prediction-LAPD-DATASET-2020-2024
Predicting Crime Case Resolutions Using Machine Learning
Project Information

This repository contains the implementation, experiments, and analysis for predicting whether reported crime cases are resolved or remain under investigation using machine learning techniques.

Dataset Used

Source: Los Angeles Police Department (LAPD)

Platform: Los Angeles Open Data Portal

Dataset: Crime Data from 2020 to Present

Link: https://data.lacity.org/Public-Safety/Crime-Data-from-2020-to-Present/2nrs-mtv8

What This Repository Contains

Exploratory Data Analysis (EDA) notebooks

Data preprocessing and feature engineering

Baseline and advanced ML models

Model evaluation and SHAP interpretability

Final academic report

Results Overview

Best Model: LightGBM

Evaluation Metric: ROC-AUC (~0.86)

Contextual and operational crime features were the strongest predictors

Future Work & Limitations

Model limitations, ethical considerations, and real-world deployment challenges

Detailed future work and recommendations are discussed in the report

How to Use

Read the report for full methodology, conclusions, and future work

Review the notebooks to understand data processing and model implementation
